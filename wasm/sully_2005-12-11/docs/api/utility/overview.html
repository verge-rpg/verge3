The utility libraries

Theses are all fairly standalone. Use them with love and for great justice.

  • Targetting Library
    This is a bunch of stuff to aid correctly seeding all the targets of an effect (be it a spell effect or an item effect or whatever), to properly send them to their resolution.

  • Inet Talk
    A very limited-use set of specific functions to demo some remedial web functionality.

  • Icons
    Just a polite little interface to ye olde icons!
