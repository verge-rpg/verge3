Welcome to VERGE 3, a versitile game-creation system designed to be powerful and easy to learn.

If you are new here, please play the packin game. Just run VERGE.EXE in this very same directory.

If you are looking for the documentation, you can find it in [the documentation folder]. [Rysen's tutorial] is a very good place to start.

If you need help after that, try asking for some on the [VERGE-RPG message Boards], or you can email me at mcgrue@verge-rpg.com or message me via AIM (screenname: astradian).

Good luck, and remember to have fun!
