Block Dodger
============

Overview
--------
Survive the onslaught of incoming blocks by evasion.
Avoid enough blocks to advance to the next level.

Controls
--------
Left/Right	Move Left or Right
B1/Enter	Retry (at Game Over)
B3/Esc		Give Up/Suicide
Alt+X		Exit
